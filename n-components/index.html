n-components

Light-weight, simple web components for use with vanilla HTML, JS and CSS.

Import the latest version of n-components into your vanilla static HTML file using:

<script src="https://unpkg.com/@trenoxxious/n-components@latest/dist/n-components.umd.js"></script>

Then, simply call any of the below n-components to get started.

You want some buttons? We have some cool, customizable n-buttons.

Base Buttons

Input:

<n-button variant="info">
More Info
</n-button>

Output:

More Info

Examples:

Normal Info Warning Alert Success

Thin Buttons

Input:

<n-button variant="warning" thin>
A warning!
</n-button>

Output:

A warning!

Examples:

Thin Thin Thin Thin Thin

Thick Buttons

Input:

<n-button thick>
Play
</n-button>

Output:

Play

Examples:

Thick Thick Thick Thick Thick

Outlined Buttons, w/ Dark Mode

Input:

<n-button outlineOnly dark thin>
Go!
</n-button>

Output:

Go!

Examples:

Outlined Outlined Outlined Outlined Outlined
